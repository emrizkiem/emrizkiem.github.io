# emrizkiem.github.io [![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

Personal website running on jekyll.

### Installing

Run local server:

```bash
git clone https://github.com/emrizkiem/emrizkiem.github.io
cd emrizkiem.github.io
bundle exec jekyll serve
```

Browse to http://localhost:4000.
